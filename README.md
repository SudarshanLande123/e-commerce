# Mini E-Commerce Website

This is a simple mini e-commerce website made using **HTML, CSS and JavaScript**.  
I built this project to practice frontend concepts like DOM manipulation, localStorage and basic UI design.

## Features
- Product listing with categories
- Search products by name
- Add to cart functionality
- Increase / decrease quantity
- Cart total, discount and final price calculation
- Light / Dark theme
- Responsive design (mobile friendly)
- Icons using Font Awesome

## Technologies Used
- HTML
- CSS
- JavaScript
- Font Awesome

## What I Learned
- How to use JavaScript with DOM
- Working with `localStorage`
- Handling cart logic (add, remove, quantity)
- Making layouts responsive using CSS
- Using icons and basic UI improvements
